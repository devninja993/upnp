# upnp
Website for UPNP


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:4200
npm run build

# build for production in local project folder
npm run build:dev

# build for production and view the bundle analyzer in folder /var/upnp.ga/dist
npm run build:prod

# run all tests
npm test
```
